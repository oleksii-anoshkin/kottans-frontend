Коротко про HTML

HTML - мова розмітки веб-документів. Набір правил для структурування (розмітки) текстової інформації, додавання зображень, створення таблиць, форм, списків тощо. Детальніше...

Для того щоб створити веб-сайт, недостатньо базових знань технології HTML. Під час розробки використовуються редактори коду, системи контролю версій, збирачі, фреймворки, бібліотеки, хостинг та інші концепції, з якими необхідно буде познайомитися.

Під час розробки веб-сайтів також використовують CSS і JavaScript. CSS - використовується для оформлення та позиціонування, відповідає за зовнішній вигляд веб-сайтів. JS - дозволяє додати функціонал, реакцію на дії користувача.

Додаткова корисна інформація

  • Символьні підстановки

    Не всі символи можна набрати на клавіатурі. Наприклад, символ зареєстрованого торгового знака & reg;, торгової марки & trade;, авторського права & copy;. для введення подібних знаків використовують символьні підстановки.

  • Зображення

    Використання графіки робить веб-сторінки візуально привабливішими. Зображення допомагають краще передати суть і зміст документа. Тег <img> призначений для розмітки зображень у різних графічних форматах.

  • Валідація

    Валідація потрібна для виявлення помилок у синтаксисі розмітки HTML-документа і розбіжностей з HTML-специфікацією, вказаною в <!DOCTYPE>. Програма для такої перевірки називається валідатором.

  • HTML Таблиці

    Таблиці дозволяють відобразити складні зв'язки, розміщуючи в клітинках будь-який контент. Їх потрібно використовувати тільки для розмітки табличних даних: розклад транспорту, календар, результати матчів, фінансові операції, меню в ресторані, прайс-лист тощо.
